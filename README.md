
# WhatsApp Chat Analysis 📊

A Python-based web application that analyzes exported WhatsApp chat data and provides meaningful insights through interactive visualizations and statistics.

## 🚀 Features

* Analyze total messages
* Count words and media shared
* Identify links shared
* Most active users
* Activity timeline (daily/monthly)
* Monthly and weekly activity
* Most common words
* Emoji analysis
* Word cloud generation
* Interactive charts and graphs

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas
* Matplotlib
* Seaborn
* Emoji
* WordCloud

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/sandeepsagar18/whatsapp-chat-analysis.git
```

2. Move into the project directory

```bash
cd whatsapp-chat-analysis
```

3. Create a virtual environment (optional)

```bash
python -m venv .venv
```

4. Activate the virtual environment

Windows:

```bash
.venv\Scripts\activate
```

Linux/macOS:

```bash
source .venv/bin/activate
```

5. Install dependencies

```bash
pip install -r requirements.txt
```

6. Run the application

```bash
streamlit run app.py
```
